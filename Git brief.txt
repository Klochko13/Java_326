0. git status - показывает состояние репозитория;
1. git init   - создает локальный Git-репозиторий в текущем каталоге;
2. git add *  - добавить все файлы в Staged-area. Все файлы в Staged-area войдут в следующий commit;
		cкрытые файлы нужно добавлять в Staged-area отдельной командой: git add .*         ;
3. git commit -m 'Commit message (сюда можно писать все что угодно)'
4. git log    - отображает историю локального репозитория;
5. git remote add origin URL(ссылка на удаленный репозиторий, который находится на GitHub) - связывает локальный репозиторий с удаленным;
6. git push origin master - отправляет локальные коммиты на удаленный репозиторий;
7. git remote remove origin - отвязывает локальный репозиторий от удаленного;
